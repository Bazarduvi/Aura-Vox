VICO NeuraPad
Neural Cognitive Interface for AI
🧬 ¿Qué es VICO NeuraPad?
NeuraPad es un sistema de interfaz cognitiva que actúa como capa de orquestación neuronal entre tú y los modelos de lenguaje cloud más avanzados del mundo.

🚀 Funciones principales:
• Conexión a modelos cloud gigantes: GPT-5, Gemini Ultra, Claude Opus, Grok y más
• Memoria neuronal persistente: aprende tu estilo, preferencias y proyectos
• Orquestación multi-agente: coordina múltiples IAs en paralelo para una tarea
• Pipeline cognitivo: cadenas de razonamiento visual + texto + código
• Integración con AURAVOX: enriquece guiones, miniaturas y audio con contexto profundo
• API unificada: un endpoint para acceder a todos los modelos cloud simultáneamente

⚡ Requiere cuenta NeuraPad activa en GitHub

⚖️ Legal & Privacidad

AURAVOX STUDIO almacena todas las configuraciones localmente en tu dispositivo. No se envían datos a servidores propios. Las claves API son procesadas directamente desde tu navegador hacia los servicios de terceros. Revisa los Términos de Servicio de cada proveedor de IA.

❓ MANUAL DE USO
🚀 INICIO RÁPIDO

1. Ve a ⚙️ Ajustes → Configura tu clave API (Gemini, Groq, Claude, etc.)
2. Selecciona la API Activa y el modelo de tu preferencia
3. Activa 💭 Modo Pensamiento Bruto para respuestas más completas
4. Usa el 🤖 Chat Agente para comenzar a crear contenido
5. Explora los módulos especializados en el menú lateral

🎬 GUIONES YOUTUBE

• Escribe el título SEO y la descripción del video
• Selecciona tipo, duración, idioma y configuración de voces
• El guión incluye marcas de tiempo [00:00], personajes, emociones e indicaciones de imágenes
• Exporta en formato YouTube, Shorts o Reels
• Usa ▶ Narrar para escuchar el guión con TTS

🎙️ AUDIO / TTS

• Usa la Web Speech API nativa del navegador (sin costo)
• Selecciona voz, idioma, velocidad, tono y emoción
• Compatible con todas las voces instaladas en tu sistema
• La visualización de onda se actualiza en tiempo real

🖼️ MINIATURAS

• Selecciona la plataforma para obtener el tamaño correcto
• Personaliza título, subtítulo, colores y estilo visual
• Descarga como PNG de alta resolución
• Tamaños: YouTube 1280×720, Instagram 1080×1080, Stories 1080×1920, etc.

📰 NOTICIAS

• Pega la URL o el texto de la noticia
• La IA resumirá y estructurará la información
• Humanizar: agrega variaciones naturales al texto
• Extender: amplía con contexto e información relacionada
• Análisis Maestro: investigación profunda con datos históricos
• Comparativa: contrasta con otras fuentes y perspectivas
• Reviews: incorpora opiniones y reacciones de la web

🌐 TRADUCCIÓN

• Usa el Chat Agente con el prompt: "Traduce el siguiente texto al [idioma]:"
• Para guiones: genera primero el guión en español y luego pide "Traduce este guión al inglés manteniendo marcas de tiempo y emociones"
• Idiomas soportados por las APIs: Español, Inglés, Portugués, Francés, Alemán, Italiano, Chino, Japonés, Árabe y más
• Usa Perplexity Sonar o Gemini 1.5 Pro para mejores traducciones técnicas
• Consejo: Activa 💭 Modo Pensamiento Bruto para traducciones más precisas con contexto cultural
• Para TTS en otro idioma: cambia el Idioma / Acento en el panel Audio antes de narrar

🔮 SUPER API FUSIÓN

• Ve a ⚙️ Ajustes → 🔮 Activar Super API Fusión
• La Super API detecta automáticamente el tipo de tarea y elige la API activa más adecuada
• Texto creativo: usa Claude o xAI Grok-3 · Código: DeepSeek Coder · Velocidad: Groq Llama instant
• Análisis web: Perplexity Sonar online · Multimodal: Gemini 2.0 Flash
• Solo funciona con APIs que tengan clave configurada y activa

🧠 xAI REASONING 4.20

• Selecciona xAI (Grok) como API activa y el modelo grok-3-mini o grok-3-mini-fast
• Activa el toggle 🧠 xAI Reasoning en la sección de Claves API
• El modelo usará reasoning_effort: high para razonamiento profundo paso a paso
• Ideal para: matemáticas, lógica, análisis complejos, código difícil, decisiones multi-paso
• Nota: en modo reasoning la temperatura se desactiva automáticamente (comportamiento esperado)

💻 MODO AGENTE / DESARROLLO PWA
